Pythonインストールガイド

タイムライン機能を使用するためにPythonをインストールしましょう

概要

タイムライン機能はFlaskというPythonフレームワークで作成されています。 この機能を使用するには、まずPythonをインストールする必要があります。

注意： ブラウザから直接Pythonをインストールすることはできません。 以下の手順に従って、お使いのOSに適した方法でインストールしてください。

Windows

1

Python公式サイトからダウンロード

https://www.python.org/downloads/ にアクセス

2

インストーラーを実行

「Add Python to PATH」にチェックを入れる

3

インストール完了

コマンドプロンプトで「python --version」で確認

macOS

1

Homebrewを使用（推奨）

brew install python
2

または公式サイトから

https://www.python.org/downloads/ からダウンロード

3

インストール確認

python3 --version

インストール後の設定

1. 必要なパッケージのインストール

タイムライン機能に必要なFlaskとその他のパッケージをインストールします：

コマンドプロンプト / ターミナル
pip install flask

2. タイムラインアプリの起動

以下のコマンドでタイムラインアプリを起動できます：

コマンドプロンプト / ターミナル
cd site/Python/timeline
python app.py

3. ブラウザでアクセス

アプリが起動したら、以下のURLにアクセス：

http://localhost:5000

トラブルシューティング

「python」コマンドが見つからない

Windowsの場合、「py」コマンドを試してください。macOSの場合、「python3」コマンドを試してください。

「pip」コマンドが見つからない

Pythonのインストール時に「Add Python to PATH」にチェックを入れ忘れた可能性があります。Pythonを再インストールしてください。

ポート5000が使用中

他のアプリケーションがポート5000を使用している可能性があります。タスクマネージャーでプロセスを確認するか、別のポートを使用してください。
